# Develop

```
# init
npm install

# build
npm run build

# incrementally build
gulp watch

```

# Cmd

```
  # Test
  npm test test/*.ts

  # Cover
  npm run cover

  # TsLint (src/**/*.ts test/**/*.ts)

  # Doc (save to docs/)
  npm doc

  # Readme (save to md/)
  npm md

```
